# LR2Nexus

OpenLR2를 위한 차세대 런처입니다.

## 소개
LR2Nexus는 OpenLR2 생태계를 위해 설계된 현대적이고 가벼운 런처입니다. 설정을 관리하고 게임을 간편하게 실행할 수 있는 효율적인 인터페이스를 제공합니다.

## 경고: 알파 릴리즈
이 버전은 **알파(Alpha) 테스트 버전**입니다. 현재 활발히 개발 중인 단계이므로 치명적인 버그, 성능 문제 또는 예기치 않은 동작이 발생할 수 있습니다. **사용으로 인해 발생하는 모든 결과에 대한 책임은 사용자 본인에게 있습니다.**

프로그램 실행 전, 아래의 중요 데이터를 반드시 백업해 두시기를 강력히 권장합니다:
* `LR2Files/Config/config.xml`
* `LR2HD/LR2files/Database/Score/<player>.db`

**참고:** LR2Nexus를 사용하여 프로필을 처음 설정할 때, 기존 LR2 프로필 비밀번호가 초기화되거나 다시 암호화될 수 있습니다. 반드시 프로필 정보를 미리 백업해 두시기 바랍니다.

## 사용 방법
1. [릴리즈 페이지](https://github.com/Unengine/LR2Nexus/releases)에서 최신 버전을 다운로드하세요.
2. ZIP 파일의 압축을 원하는 폴더에 풉니다. (LR2 본체와 같은 경로에 둘 필요는 없습니다.)
3. `LR2Nexus.exe`를 실행합니다.
4. 첫 실행 시, OpenLR2 실행 파일의 경로를 지정합니다.
5. 설정이 완료되었습니다!
